## Projects Overview

### 1. Linear Regression
**Notebook:** `01_Linear_Regression_Basics.ipynb`

Introduction to supervised learning with Linear Regression using scikit-learn. Demonstrates model fitting, prediction, and visualization of regression lines.

**Key Concepts:**
- Model fitting with `LinearRegression()`
- Data reshaping for sklearn compatibility
- Prediction and visualization

---

### 2. Data Visualization
**Notebooks:** `02-04_Data_Visualization_*.ipynb`

Comprehensive exploration of data visualization techniques using Matplotlib and Pandas for exploratory data analysis.

**Techniques Covered:**
- Bar charts for categorical data
- Scatter plots for relationship analysis
- Line plots for trend visualization
- Combined visualization approaches

---

### 3. Random Forest Classification
**Notebooks:** `05-12_Random_Forest_Classification_*.ipynb`

Multiple implementations of Random Forest Classifier for various classification tasks, demonstrating the complete ML pipeline.

**Workflow:**
1. **Data Loading** - Reading CSV datasets with Pandas
2. **Exploratory Data Analysis** - Using `head()`, `tail()`, `shape`, `info()`, `isnull()`
3. **Data Preprocessing** - Feature and target separation
4. **Train-Test Split** - Using `train_test_split()` from sklearn
5. **Model Training** - `RandomForestClassifier()` implementation
6. **Evaluation** - Accuracy score calculation

---

### 4. Diabetes Prediction
**Notebook:** `13_Diabetes_Prediction_RandomForest.ipynb`
**Script:** `scripts/diabetes.py`

A complete machine learning project for predicting diabetes using the Pima Indians Diabetes Dataset with Random Forest algorithm.

**Features:**
- Binary classification (Diabetic/Non-Diabetic)
- 80-20 train-test split
- Accuracy evaluation metrics
